# SIT162 Digital Folio (User Interfaces And Mobile Applications)
Simple UI App for IOS and WatchOS made for SIT162 Assignment 3 following on from this video tutorial https://www.youtube.com/watch?v=09TeUXjzpKs
# Aim
the aim/challenge for this section was to create an application that allowed the user to create a social media post as fast as possible.
All the project files and images for Xcode 14 are located in the master branch. simply download, unzip and run the EasyPost.xcodeproj file to build the project.
